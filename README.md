# 每日重大新闻

一个分为“国内”和“国外”两个模块的新闻看板。页面打开时会加载一次新闻，之后只有用户点击“刷新新闻”按钮才会重新拉取。

## 本地预览

直接用本地静态服务打开即可，例如：

```bash
python -m http.server 5173
```

然后访问：

```text
http://127.0.0.1:5173/
```

## 部署到 Vercel

把本目录上传到 GitHub 仓库，然后在 Vercel 里选择 `New Project`，导入这个仓库并点击 `Deploy`。

部署后，网站会优先请求：

```text
/api/news
```

这个接口由 Vercel 云端执行，会抓取 RSS 和新闻正文里的图片，再返回给前端。这样比浏览器直接抓新闻更稳定，也能更好满足“每条新闻封面必须来自对应新闻内容”的要求。

## 文件说明

- `index.html`：页面结构
- `styles.css`：页面样式
- `app.js`：前端交互、搜索、手动刷新、浏览器兜底抓取
- `api/news.js`：Vercel 云端新闻接口
